# Quotes Web Scraper
This is a simple Python project for learning web scraping.  
It extracts quotes, authors, and tags from [quotes.toscrape.com](https://quotes.toscrape.com) using the Python libraries `requests` and `beautifulsoup4` ,then  saves them to a CSV file.

## Features
- Scrapes all pages automatically
- Extracts quote text, author name, and tags
- Saves data to CSV
