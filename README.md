# Quotes Web Scraper
This is a simple Python project for learning web scraping.  
It extracts quotes, authors, and tags from [quotes.toscrape.com](https://quotes.toscrape.com) using the Python libraries `requests` and `beautifulsoup`, then  saves them to a CSV file using a `CSV` library.

## What it does
- Scrapes all pages automatically
- Extracts quote text, author name, and tags
- Saves data to CSV
