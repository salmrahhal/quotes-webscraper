# Quotes Web Scraper
This Python project scrapes quotes, authors, and tags from [quotes.toscrape.com](https://quotes.toscrape.com) and saves them into a CSV file.

## Features
- Scrapes all pages automatically
- Extracts quote text, author name, and tags
- Saves data to CSV with UTF-8 encoding
- Handles special characters and fancy quotes

## Usage
2. Run `python scraper.py`
3. Find the output CSV in the project folder
